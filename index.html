Amanzi

Online arm editor and instruction set simulator and debugger

Download Download View Amanzi on GitHub

Project for CO227 in the University of Peradeniya Faculty of Engineering Department of Computer Engineering.

In here are the source files of an Online ARM instruction set simulator and debugger which was made by the group ARM3 in the course CO227 codenamed Amanzi, the server requires to be linux which supports gcc-arm-linux-gnueabi and qemu. This has been tested on Google Chrome and Mozilla Firefox. Yet further testing and releases are on it's way.
